Project Description 

This is a simple, console-based Number Guessing Game implemented in Python. The program selects a random number within a specified range, and the user must guess the number. The game provides hints (too high/too low) after each incorrect attempt.


This project was built to demonstrate proficiency in basic programming logic and the use of modern development tools, including Visual Studio Code, Git, and GitHub.



Features 

Random Number Generation: The secret number is randomly generated on each run.

Attempt Counter: Tracks and reports the number of guesses the user takes.

Feedback System: Provides "Too High" or "Too Low" hints to guide the user.

Simple Console Interface: Easy-to-use text-based interaction.

Installation and Usage Instructions 

Prerequisites
You need Python 3.x installed on your system to run this game.

Installation
Clone the Repository: Open your terminal or command prompt and clone this repository to your local machine:

Bash

git clone https://github.com/YourUsername/YourRepoName.git
Navigate to the Project Directory:

Bash

cd YourRepoName
Usage
Run the Game: Execute the main Python script:

Bash

python main.py
Play: Follow the on-screen prompts to enter your guesses.
